二、串联指令

Dockerfile 中最多只能有 127 个 RUN 指令，镜像制作中我们可以将多条 RUN 指令合并为一条 RUN 指令。

看看串联后构建的镜像大小：

可明显看到，redis:v3 镜像的大小仅为 144M，是没有将 RUN 指令合并前构建镜像的二分之一还要小，可见 RUN 指令串联后效果是很明显的。

三、压缩镜像
